# KPI 정의 및 계산 기준

## 계산식

| KPI | 계산식 | 분자 소스 | 분모 소스 |
|---|---|---|---|
| CTR | 클릭 / 노출 | 채널 | 채널 |
| CPI | 비용 / 앱설치 | 채널 | AppsFlyer |
| CPA_회원가입 | 비용 / 회원가입 | 채널 | AppsFlyer |
| CPA_구매 | 비용 / 구매 | 채널 | AppsFlyer |
| ROAS | 구매매출 / 비용 | AppsFlyer | 채널 |

## 기준선 (임시)

| KPI | 양호 | 위험 |
|---|---|---|
| ROAS | ≥ 4.0 | < 2.0 |

## 공통 규칙

- 분모가 0인 경우 결과는 `NULL`로 처리 (`NULLIF(분모, 0)` 사용). `0`이나 `inf` 금지.
- ROAS 소수점은 2자리까지 반올림 (`ROUND(..., 2)`).
- 비용·매출 단위는 **원(₩)** 기준. 외부 공유 시 단위 변환 필요 여부는 별도 협의.
