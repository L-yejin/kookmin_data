# 캠페인 구조 정의

## 캠페인 네이밍

```
{채널약어}_CMP_{번호}_{캠페인목적}

예) GGL_CMP_01_플러스가입
    META_CMP_03_리타겟팅
```

## 캠페인 목적

| 목적 | 설명 | 분석 시 주의 |
|---|---|---|
| 플러스가입 | 멤버십 신규 가입 유도 | |
| 첫구매 | 첫 구매 전환 | |
| 신규유저 | 앱 신규 설치 유저 획득 | |
| 룩얼라이크 | 기존 고객 유사 타겟 확장 | |
| 리타겟팅 | 기존 방문자 재유입 | **신규 캠페인과 ROAS 분리 비교** |
| 재구매 | 기존 구매자 반복 구매 유도 | **신규 캠페인과 ROAS 분리 비교** |
| 브랜드KW | 브랜드 키워드 검색 광고 | 네이버 전용 |
| 일반KW | 일반 키워드 검색 광고 | 네이버 전용 |

## 타겟 그룹

| 그룹명 | 설명 |
|---|---|
| 논타겟 | 타겟팅 없는 일반 노출 |
| 유사타겟 | 기존 고객 유사 타겟 (LAL) |
| 리마케팅 | 앱 방문 이력 있는 유저 |
| VIP | 고가치 고객 세그먼트 |
| 윈백 | 이탈 후 재유입 대상 |

## 분석 원칙

- `리타겟팅` · `재구매` 목적 캠페인은 신규 캠페인과 **별도 집계**한다.
  모수(기존 고객)가 다르기 때문에 ROAS 단순 비교 시 왜곡된다.
- `브랜드KW` · `일반KW` 는 입찰 방식이 달라 노출·클릭 성격이 다르므로 디스플레이 캠페인과 분리.
