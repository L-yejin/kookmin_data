# 데이터 소스 정의

## 소스별 역할

### channel.csv — 광고 집행 기준 데이터
매체 플랫폼에서 직접 추출한 광고 성과 데이터.

| 컬럼 | 설명 |
|---|---|
| 일 | 집계 날짜 (YYYY-MM-DD) |
| 채널 | 광고 채널 (구글/메타/네이버) |
| 채널분류 | 외부/자체 |
| 캠페인 | 캠페인명 |
| 캠페인목적 | 플러스가입/리타겟팅 등 |
| 그룹 | 타겟 그룹명 |
| 소재 | 소재명 (네이밍 컨벤션 참고) |
| 노출 | 광고 노출 수 |
| 클릭 | 광고 클릭 수 |
| 비용 | 광고 집행 비용 (₩) |
| 회원가입 | 채널 기준 회원가입 전환 수 |
| 구매 | 채널 기준 구매 전환 수 |
| 구매매출 | 채널 기준 구매 매출 (₩) |

### appsflyer.csv — 앱 기여(Attribution) 데이터
AppsFlyer SDK 기반 모바일 기여 데이터. 채널 데이터보다 실제 앱 내 행동에 가깝다.

| 컬럼 | 설명 |
|---|---|
| 일 | 집계 날짜 (YYYY-MM-DD) |
| 미디어소스 | AF 채널 식별자 (채널 매핑 문서 참고) |
| 캠페인 | 캠페인명 |
| 그룹 | 타겟 그룹명 |
| 소재 | 소재명 |
| 클릭 | AF 기준 클릭 수 |
| 회원가입 | AF 기준 회원가입 전환 수 |
| 구매 | AF 기준 구매 전환 수 |
| 구매매출 | AF 기준 구매 매출 (₩) |

## Truth Source 원칙

| 지표 | 사용 소스 | 이유 |
|---|---|---|
| 노출 | 채널 | AF에 노출 데이터 없음 |
| 클릭 | 채널 | 비용 계산 기준 |
| 비용 | 채널 | 실제 청구 기준 |
| 앱설치 | AppsFlyer | SDK 직접 측정 |
| 회원가입·구매·매출 | AppsFlyer | Attribution 기준, 중복 제거됨 |

## 이상치 감지 기준

채널 클릭 수와 AF 클릭 수 차이가 **20% 초과** 시 이상치 플래그 처리.

```python
# DuckDB 예시
ABS(ch.클릭 - af.클릭) / NULLIF(ch.클릭, 0) > 0.2 AS 클릭_이상치
```
